DevVault = your personal developer dashboard. think of it like a mini Notion built for developers.
when it's done, a user can:

create an account / log in — their own private space
write and save notes/snippets — like code notes, ideas, whatever
create tasks/todos — track what you're working on
save bookmarks — useful links, docs, resources
upload files — images, code files, whatever
track coding goals — "learn Express by June" type stuff
search everything — across all their notes, tasks, bookmarks
switch dark/light theme
see a dashboard — analytics, recent activity, overview

basically imagine you log in and see YOUR stuff. your notes, your tasks, your goals. nobody else's.

the reason this project is good is because building it naturally forces you to learn:

how servers work
how databases store user data
how login systems actually function
how files get uploaded
how to structure real backend code
